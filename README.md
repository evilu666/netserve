# Netserve
## Installation
```bash
> python -m venv .env
> pip install -r requirements.txt 
```

## Usage
### Starting server
```bash
> source .env/bin/activate # Only needed once per terminal session
> python server.py
```

### Using client
Refer to command-line help
```bash
> source .env/bin/activate # Only needed once per terminal session
> python client.py --help
```
